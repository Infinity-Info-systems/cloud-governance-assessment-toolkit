# Use-Case Notes

## Common Assessment Scenarios

- enterprise cloud governance review
- platform reliability maturity check
- disaster recovery readiness review
- FinOps operating model assessment
- security control evidence review

## Typical Outputs

- scorecard summary
- executive brief
- remediation roadmap
- evidence appendix

## Reuse Guidance

- adapt the same structure for each business unit
- keep one evidence register per assessment cycle
- compare trend over time rather than isolated snapshots

## Use

Use these notes to keep recurring assessments comparable and easy to explain to stakeholders.
